# Product Overview

Astro ベースの技術ブログポートフォリオ。高速なパフォーマンスを維持しながら、個人の技術的な知見や学習記録を発信するプラットフォーム。

## Core Capabilities

- **Static Site Generation (SSG)**: Core Web Vitals を重視した高速な静的サイト生成
- **MDX サポート**: Markdown + React コンポーネントによる豊富な記事表現
- **OG Image 自動生成**: 記事ごとの OG 画像を動的に生成
- **Vercel デプロイ**: シームレスな CI/CD とホスティング
- **Island Architecture**: 必要な部分のみを動的に Hydration する部分的インタラクティブ化

## Target Use Cases

- 技術記事の執筆と公開
- 個人のポートフォリオとしての情報発信
- 学習記録のアーカイブ
- 今後の拡張: AI チャット機能（SF6 パッチノート RAG など）による付加価値提供

## Value Proposition

- **パフォーマンス最優先**: 静的生成による高速レスポンス
- **開発者体験**: TypeScript + Astro によるモダンな開発環境
- **拡張性**: Island Architecture による段階的な機能追加
- **コスト効率**: Serverless + 無料枠活用による持続可能な運用
